# js-sort
This npm package contains a few sorting algorithms to sort arrays of numbers.

## Quicksort
```js
const { quicksort } = require('js-sort');

const arr = quicksort([3, 1, 2]);
```

## Bubblesort
```js
const { bubblesort } = require('js-sort');

const arr = bubblesort([3, 1, 2]);
```
